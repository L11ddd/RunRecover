# Demo Cases

The web app includes three one-click cases for the 5-minute MVP demo.

## Easy Recovery

- 5 km easy run
- RPE 3
- 8 hours sleep
- Fatigue 2, soreness 2
- Expected: light recovery, normal hydration and meals, tomorrow can be normal or easy activity.

## High-Intensity Night Run

- 8 km tempo run at night
- RPE 8
- 5.8 hours sleep
- Fatigue 7, soreness 5
- Tomorrow intensity plan
- Expected: key recovery, RPE and sleep appear in reasons, tomorrow intensity is discouraged.

## Long High-RPE Run

- 16 km long run
- RPE 9
- 6 hours sleep
- Fatigue 8, soreness 7
- Expected: high-load reminder, tomorrow rest or walking, conservative recovery language.
