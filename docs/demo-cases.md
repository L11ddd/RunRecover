# Demo Cases

The web app includes three one-click cases for the 5-minute MVP demo.

## Easy Recovery Run

- 4 km recovery run
- User level: beginner
- Modifier: completed
- Recent 48 hours: rest
- RPE 2
- 8.2 hours sleep
- Fatigue 2, soreness 1
- Expected: light recovery, normal hydration and meals, tomorrow can be normal or easy activity.

## Sleep-Debt Steady Run

- 9 km steady run with hills at night
- User level: regular
- Recent 48 hours: easy training
- RPE 6
- 4.8 hours sleep
- Fatigue 6, soreness 3
- Tomorrow strength or cross-training plan
- Expected: key recovery, mainly driven by sleep debt, night timing, accumulated fatigue, and tomorrow-plan pressure.

## Long Muscle-Load Run

- 18 km long run with a target-pace block
- User level: advanced
- Recent 48 hours: easy training
- RPE 7
- 7.4 hours sleep
- Fatigue 7, soreness 8
- Tomorrow rest plan
- Expected: high-load reminder, driven by distance, duration, long-run type, and soreness rather than sleep debt.
